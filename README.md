# Three_column_card
My challenge was to build a 3-column card component that matches the provided design.
This project is to test my understanding of CSS layout techniques, specifically flexbox, and your ability to create visually consistent components with proper spacing and styling.

## Features
Build a fixed 3-column card component using only HTML and CSS that displays SEDAN, SUV, and LUXURY vehicle categories side by side with distinct background colors, icons, descriptive text, and a ** learn more ** button.
Use flexbox with ** display: flex and align-items: stretch ** to create equal-height cards in a single row, with no responsive breakpoints required.
