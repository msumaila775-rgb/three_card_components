# Three_column_card
My challenge was to build a 3-column card component that matches the provided design.
This project is to test my understanding of CSS layout techniques, specifically flexbox, and your ability to create visually consistent components with proper spacing and styling.

## Features
Build a fixed 3-column card component using only HTML and CSS that displays SEDAN, SUV, and LUXURY vehicle categories side by side with distinct background colors, icons, descriptive text, and a learn more  button.
Use flexbox with display: flex and align-items: stretch to create equal-height cards in a single row, with no responsive breakpoints required.
Style each card with proper border-radius on the outer corners only (rounded left on first card, rounded right on last card), and add hover effects on buttons that swap the background and text colors with smooth transitions.

### HTML and CSS
I first created a main element with a class container then created an article element with a class sedan-card.
Inside the article element, I added an img icon with the sedan icon downloaded from svg, a h1 element with a text "SEDAN", a p element with its descriptions and a button.
After that, I copied and pasted the article element but changed the img icon, h1 and p to a suv icon, SUV, its description and luxury icon, LUXURY and its description respectively.
For the CSS I created appropriate styles for the elements and class.
